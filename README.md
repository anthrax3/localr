# Localr
 :computer: Map localhost ports to localr.io


# Install

Install using `npm`:

```
npm install localr -g
```

# Getting Started

```
localr add 8080 webserver.localr.io
```

and then browse `http://webserver.localr.io`. Easy, isn't it?
